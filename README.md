## Installation

```bash
npm install @zitterorg/aspernatur-qui-recusandae
```

## Usage

```javascript
var isLetter = require('@zitterorg/aspernatur-qui-recusandae');

isLetter('a'); // => true
isLetter('abba'); // => false
```
